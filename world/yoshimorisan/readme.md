targetObject: 回転させるオブジェクト

baseSpeed:基本の回転速度

このスクリプトを適当なオブジェクトにアタッチしてください。
オブジェクトをOFFにすると動作しません。必要に応じてHierarchyからGameCubeを作成し、そこにアタッチしてください

上記の値は適当にデフォルトになってるので状況に合わせて変更してください
targetObjectの箇所に動かしたいオブジェクトをD＆Dしてください
